# Sistema de Manipulação de Matrizes Esparsas em C

Projeto desenvolvido em linguagem C para a disciplina de Estrutura de Dados, com foco na implementação de matrizes esparsas utilizando listas simplesmente encadeadas e alocação dinâmica de memória.  

O sistema permite criar e manipular múltiplas matrizes esparsas, armazenando apenas os elementos diferentes de zero, reduzindo desperdício de memória e melhorando a eficiência no armazenamento de dados.  

## • Funcionalidades

▸ Criação de múltiplas matrizes

▸ Definição dinâmica de dimensões

▸ Inserção de elementos não nulos

▸ Busca de valores por linha e coluna

▸ Impressão completa da matriz

▸ Soma, Subtração e Multiplicação de Matrizes

▸ Geração da matriz transposta

▸ Exibição da diagonal principal

▸ Liberação dinâmica da memória utilizada

## • Principais Conceitos Aplicados:

▸ Matrizes esparsas

▸ Listas encadeadas

▸ Ponteiros

▸ Structs

▸ Alocação dinâmica com malloc

▸ Percurso de listas

▸ Modularização com funções

▸ Operações matriciais

## • Estrutura Utilizada
Cada elemento não nulo da matriz é armazenado em um nodo contendo:

▸ valor do elemento

▸ linha

▸ coluna

▸ ponteiro para o próximo nodo

## • Tecnologias
▸ Linguagem C

 • O principal objetivo do projeto foi praticar conceitos de Estrutura de Dados, manipulação de memória dinâmica e operações matemáticas utilizando estruturas encadeadas em linguagem C.
